Dain Store adalah aplikasi e-commerce fashion yang dibangun menggunakan Laravel 12.40.2 untuk mengelola produk pakaian, data member, dan laporan penjualan dengan fitur Role-Based Access Control (Admin dapat CRUD product & member, Owner hanya bisa view). Sistem ini dikembangkan dengan MySQL di XAMPP localhost dan di-host di GitHub, menggunakan Laravel Breeze untuk authentication dan Sanctum untuk API dengan 55+ unit test & feature test yang telah lulus validasi, mendukung RESTful API, Dashboard real-time, Product Management dengan upload gambar, Member Management, dan Reporting System yang production-ready.
